## üìÅ File System Commands

| Command | Description |
|-------|------------|
| `ls` | Lists files and folders in the current directory |
| `ls -l` | Shows files and folders with detailed information |
| `ls -a` | Shows all files including hidden files |
| `pwd` | Shows the current working directory |
| `cd dir` | Changes to the specified directory |
| `cd ..` | Moves one level up to the parent directory |
| `mkdir` | Creates a new directory |
| `rmdir` | Deletes an empty directory |
| `touch` | Creates an empty file |
| `cat` | Displays the content of a file |
| `cp` | Copies a file or directory from source to destination |
